## 1. 产品概述

药房慢病随访管理系统——帮助药师根据处方有效期、购药周期、慢病指标、历史反馈和漏报次数智能判断重点跟进患者，支持药师建立随访计划并分组、患者提交用药反馈、店长查看逾期名单和督办情况。系统覆盖过敏禁忌检查、隐私授权管理、复诊建议、批量提醒、药品补货预警和停药归档，保留完整业务轨迹。

- 目标用户：社区药房药师、慢病患者、药房店长
- 核心价值：降低慢病患者失访率，提升复诊依从性，减少用药安全风险

## 2. 核心功能

### 2.1 用户角色

| 角色 | 进入方式 | 核心权限 |
|------|----------|----------|
| 药师 | 角色切换进入 | 创建/编辑随访计划、患者分组、提交反馈（代患者）、批量提醒、药品补货预警 |
| 患者 | 角色切换进入 | 提交用药反馈、查看随访计划、查看复诊建议 |
| 店长 | 角色切换进入 | 查看逾期名单、督办跟进、查看业务轨迹、停药归档审核 |

### 2.2 功能模块

1. **工作台**：关键指标卡片（待随访、逾期、重点关注、停药归档）、近期随访日历、待办事项、预警汇总
2. **随访计划**：创建/编辑计划、患者分组（按病种/风险等级/购药周期）、处方有效期跟踪、自动生成提醒
3. **患者管理**：患者档案、用药反馈提交、过敏禁忌标记、隐私授权状态、复诊建议、停药操作与归档
4. **逾期督办**：逾期名单列表、督办状态跟踪、连续未反馈升级、处方过期拦截提示
5. **业务轨迹**：计划生成、反馈变更、逾期升级、处方过期拦截、停药后停止提醒的操作日志

## 3. 核心流程

### 3.1 随访计划生成流程
药师创建随访计划 → 系统检查患者过敏禁忌 → 检查隐私授权 → 关联处方有效期 → 按购药周期自动生成提醒时间节点 → 加入对应分组 → 记录"计划生成"轨迹

### 3.2 用药反馈流程
到达随访时间 → 药师/患者提交反馈 → 系统检查过敏禁忌冲突 → 记录反馈 → 判断是否连续两次未反馈 → 若是则升级为重点关注 → 检查处方是否过期 → 过期则拦截并生成复诊建议 → 记录"反馈变更"或"逾期升级"轨迹

### 3.3 逾期督办流程
随访超时未反馈 → 系统自动标记逾期 → 按逾期天数排序进入逾期名单 → 店长查看督办 → 标记督办状态 → 连续两次未反馈自动升级重点关注 → 记录"逾期升级"轨迹

### 3.4 停药归档流程
药师确认患者停药 → 系统停止生成随访提醒 → 患者进入停药归档 → 后续不再生成随访 → 记录"停药后停止提醒"轨迹

### 3.5 处方过期拦截流程
处方到达有效期 → 系统拦截后续随访 → 提示需复诊 → 药师编辑复诊建议 → 通知患者 → 记录"处方过期拦截"轨迹

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：深青色（teal）#0D9488，传达医疗专业感与信任感
- 辅助色：琥珀色（amber）#F59E0B 用于预警/逾期，玫红（rose）#E11D48 用于紧急/过敏
- 中性色：石板灰（slate）系列用于背景和文字
- 按钮风格：圆角微阴影，主操作用实色填充，次操作用描边
- 字体：Noto Sans SC（中文）+ DM Sans（数字/英文），层次分明
- 布局：左侧固定侧边栏导航 + 右侧内容区，卡片式模块化布局
- 图标：Lucide React 图标库，线条风格

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 工作台 | 指标卡片 | 4列网格，每卡带图标+数字+趋势箭头，hover 微上浮 |
| 工作台 | 预警汇总 | 紧凑列表，红/黄/蓝标签区分紧急程度 |
| 工作台 | 近期待办 | 时间轴样式，按日期分组 |
| 随访计划 | 计划列表 | 表格+分组折叠，行内操作按钮 |
| 随访计划 | 新建计划 | 侧滑抽屉面板，表单分步骤 |
| 随访计划 | 批量提醒 | 弹窗确认，显示勾选患者数量 |
| 患者管理 | 患者列表 | 卡片网格+列表视图切换，风险等级色标 |
| 患者管理 | 患者详情 | 右侧抽屉，多标签页（档案/反馈/处方/过敏/隐私） |
| 患者管理 | 用药反馈 | 模态框表单，含疗效评分滑块 |
| 患者管理 | 停药归档 | 确认弹窗+归档原因选择 |
| 逾期督办 | 逾期名单 | 表格，逾期天数热力色标，行内督办操作 |
| 逾期督办 | 处方过期拦截 | 过期行高亮红色底色+复诊按钮 |
| 业务轨迹 | 轨迹列表 | 时间轴样式，不同类型用不同颜色圆点 |

### 4.3 响应式
- 桌面优先设计（1440px+ 最佳体验）
- 平板（768px-1024px）：侧边栏折叠为图标模式
- 手机（< 768px）：底部标签栏导航，卡片单列布局

### 4.4 交互细节
- 角色切换：顶部导航栏右侧下拉选择，切换后侧边栏菜单和页面权限联动
- 预警通知：页面右上角铃铛图标，红点提示数量
- 列表操作：行内按钮 + 右键菜单
- 确认操作：删除/停药/批量操作需二次确认弹窗
